# 🚀 Hasnain’s Python Projects Collection  

Welcome to my Python projects repository! This repository contains a series of mini-projects showcasing fundamental concepts, logic building, and problem-solving skills. Each project explores a unique topic and is designed to strengthen coding skills using Python.  

---

## 📚 **Project List**  

### 1. **Countdown Program**  
- **Description**: Prints a countdown from 10 to 1, followed by "Lift Off!"  
- **Concepts Covered**: For loops and dynamic printing.  
- **Notebook**: [Countdown Program](./Countdown.ipynb)

---

### 2. **Guess My Number**  
- **Description**: A guessing game where the user guesses if a randomly selected number is high or low.  
- **Concepts Covered**: Conditional logic, loops, and input handling.  
- **Notebook**: [Guess My Number](./GuessMyNumber.ipynb)

---

### 3. **Random Number Generator**  
- **Description**: Generates 10 random numbers between 1 and 100.  
- **Concepts Covered**: For loops and the `randint` function.  
- **Notebook**: [Random Number Generator](./RandomNumberGenerator.ipynb)

---

### 4. **Advanced Guessing Game**  
- **Description**: A number guessing game with multiple versions, including OOP and user inputs for name, age, and rounds.  
- **Concepts Covered**: Error handling, classes, and user interaction.  
- **Notebook**: [Advanced Guessing Game](./AdvancedGuessingGame.ipynb)

---

### 5. **Weight on Planets**  
- **Description**: Calculates weight on different planets based on Earth weight.  
- **Milestones**:  
  - **Milestone 1**: Calculates weight only on Mars.  
  - **Milestone 2**: Calculates weight on any planet, with proper input handling.  
- **Notebook**: [Weight on Planets](./WeightOnPlanets.ipynb)

---

### 6. **List Operations**  
- **Description**: Demonstrates advanced list operations including adding, modifying, accessing, and slicing list elements.  
- **Concepts Covered**: `.append`, custom functions, and indexing.  
- **Notebook**: [List Operations](./ListOperations.ipynb)

---
